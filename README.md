# Applicant Form Plugin

## First
-  `git clone git@github.com:nayanchamp7/applicant-form.git`
- `npm install`
- `npm i -g gulp-cli`
- `gulp build`

## Add Shortcode:

- Copy and add this shortcode `applicant_form` to display the form (like below)
